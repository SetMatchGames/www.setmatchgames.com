Set Match Games

Video games designed by their players.

The best multiplayer games of the last two decades were designed by players. MOBAs, Counter-Strike, and Battle Royals all started out as mods.

Set Match Games is building a platform that fully embraces community-generated content, from game design and balance to art assets and profit-sharing.

Decisions about games will be made by their communities, using curation markets and other forms of decentralized governance. Game clients will use gossip protocols to connect players securely without any central servers. We'll start simply with card games, but eventually, we hope the Set Match Games ecosystem will support games of all genres and sizes.

Set Match Games is an open-source project. If you're interested in particpating, please reach out. We respond to all messages on Twitter, and we stream our development process on Twitch every Saturday from 1-3 PM EST.
